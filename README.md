Student Portfolio and Academic Management Website

Project Overview

This project is a responsive multi-page website developed as part of a university web development assignment. The website serves as both a personal academic portfolio and a simple academic management platform for a university student. It showcases personal information, technical skills, academic records, projects, and contact details while demonstrating the use of HTML, CSS, and JavaScript.

Features

- Responsive design for desktop, tablet, and mobile devices
- Multi-page website
- Home page with student profile and biography
- About page with personal and academic information
- Skills page displaying technical and professional skills
- Academic Management page with course registration and timetable
- Projects page showcasing completed and future projects
- Contact page with a contact form and social media links
- Responsive navigation menu
- Scroll animations and interactive effects
- Dark mode support
- Back-to-top button
- Smooth scrolling

---

Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Font Awesome Icons
- Google Fonts (Poppins)

1. Download or clone the repository.
2. Open the project folder.
3. Open "index.html" in your preferred web browser.

Alternatively, run the project using the Live Server extension in Visual Studio Code for the best development experience.
